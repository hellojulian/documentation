## How It Works

1. **Figma Detection**: The sync script automatically detects component frames in your Figma file
2. **Screenshot Capture**: High-resolution screenshots are generated for each component
3. **Documentation Update**: This page and individual component pages are updated with the latest images
4. **Version Control**: Changes are tracked in git with automatic PR creation
