## Features

<AccordionGroup>
  <Accordion title="Automated Sync">
    Changes in Figma automatically trigger documentation updates via webhooks
  </Accordion>

  <Accordion title="Visual Diffs">
    See exactly what changed with automatic screenshot comparisons
  </Accordion>

  <Accordion title="Live Design Tokens">
    Design tokens are pulled directly from Figma Variables and kept in sync
  </Accordion>

  <Accordion title="Interactive Prototypes">
    Embed live Figma prototypes directly in your documentation
  </Accordion>
</AccordionGroup>

## Getting Started

1. **Connect Figma**: Set up your Figma API token and file URLs
2. **Configure Webhooks**: Enable real-time sync when designs change
3. **Deploy**: Your documentation automatically updates and deploys
